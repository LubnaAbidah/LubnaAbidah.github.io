Aplikasi Pemesanan Tiket Travel Berbasis Web

Aplikasi Pemesanan Tiket Travel Berbasis Web merupakan web yang menyediakan layanan pemesanan tiket secara online. Dalam pembuatan aplikasi tersebut terdapat 4 tim dimana terdapat bagian analisis desain database 1 orang, programmer 2 orang, dan UI/UX desainer 1 orang. Saya bertanggung jawab pada pembuatan program bagian admin dan operator. User dapat melakukan register dan melakukan pemesanan tiket dan mencetak tiket. Admin dapat melakukan update status pemesanan tiket user dan pengolahan data laporan pendapatan serta menyimpan dalam bentuk PDF. Operator dapat melihat data pemesanan dan update data user. Proyek ini dikerjakan menggunakan kolaborasi antara input process output dengan PHP dan MySQL serta menggabungkan sistem dengan desain web yang menggunakan tema Bootstrap.
